== Description ==

The plugin is used for config the MaanTheme mobile and support RestAPI to connect to the app.

[youtube https://youtu.be/hTtrRJAEC20]

MaanTechnology focuses on building modern useful web templates as well as mobile application for solving modern problems. As a part of it, different mobile apps require functionality to communicate with websites. As a solution, this plugin MaanStore API is created. It is specially built for communication between the mobile application and the website using REST API.

== Installation ==

= Manual Installation =

1. Upload the entire `/maanstore-api` directory to the `/wp-content/plugins/` directory.
1. Activate 'MaanStore API' through the 'Plugins' menu in WordPress.

= Better Installation =

1. Go to Plugins > Add New in your WordPress admin and search for 'MaanStore API'.
1. Click Install.
